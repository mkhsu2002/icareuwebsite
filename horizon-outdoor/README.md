# Horizon Outdoor 品牌授權網站

這是一個 Horizon Outdoor 品牌授權合作夥伴招募的網站，使用 React + TypeScript + Vite 建構。

## 功能特色

- 響應式設計，支援各種裝置
- SEO 優化，包含 meta 標籤、結構化資料、sitemap
- 品牌故事與理念展示
- 合作夥伴類型介紹
- 產品線展示
- 聯絡表單整合

## 技術棧

- React 19.2.0
- TypeScript
- Vite 6.2.0
- Tailwind CSS
- GitHub Pages 部署

## 開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建構專案
npm run build

# 預覽建構結果
npm run preview
```

## 部署

### GitHub Pages 自動部署

1. 將程式碼推送到 GitHub 儲存庫
2. 在 GitHub 儲存庫設定中啟用 GitHub Pages
3. 選擇 "GitHub Actions" 作為來源
4. 每次推送到 main 分支時會自動部署

### 手動部署

```bash
# 建構並部署到 GitHub Pages
npm run deploy
```
